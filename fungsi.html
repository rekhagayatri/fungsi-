Sejarah fungsi

Konsep fungsi tepat dianggap sebagai salah satu yang paling penting dalam semua matematika. Sebagai titik, garis dan teori yang dominan dari zaman Yunani Kuno sampai Zaman Modern. Pengertian fungsi dan turunan merupakan dasar dari analisis matematika, teori yang menjadi pusat dalam pengembangan matematika sejak saat itu. Secara historis, beberapa ahli matematika dapat dianggap telah diramalkan dengan formulasi modern konsep fungsi.

Di antara mereka adalah Oresme (1323-1382), yang mengembangkan teori geometris lintang bentuk yang mewakili tingkat intensitas yang berbeda dan ekstensi. Munculnya gagasan fungsi sebagai entitas matematika individual dapat ditelusuri ke awal dari kalkulus. Descartes (1596-1650) dengan jelas menyatakan bahwa persamaan dalam dua variabel, geometris diwakili oleh kurva yang menunjukkan ketergantungan antara jumlah variabel.

fungsi dan nilai fungsi

• Suatu fungsi dari A ke B (dimana A dan B bukan merupakan huimpunan kosong) adalah aturan yang memasangkan setiap anggota A dengan tepat satu anggota B

• Himpunan A disebut domain atau daerah asal

• Himpunan A disebut domain atau daerah asal

• Himpunan B disebut kodomain atau daerah kawan

• Himpunan semua peta di B disebut range

b. Himpunan pasangan berurut

Fungsi sebagai pasangan berurut dari dua bilangan real x dan y adalah himpunan (x,y) dengan x ∈ D f paling banyak muncul satu kali dalam setiap pemetaan dan x harus mempunyai pasangan kodomain (K f)

c. grafik fungsi

Nilai fungsi

Jika f menyatakan fungsi pada x suatu masukan untuk fungsi f maka nilai outputnya ditulis f (x) yang berarti f dan x.

Contoh:

f(x) = 7x – 4

tentukanlah nilai fungsi untuk x = 2

Jawab:

f(x) = 7x – 4

f(2) = 7.2 – 4

f(2) = 14 – 4

f(2) = 10

Jadi f(2) adalah 10.

Macam-Macam Fungsi

Dalam matematika sering dijumpai berbagai macam fungsi, diantaranya mempunyai ciri-ciri khusus. Fungsi yang memiliki ciri-ciri khusus disebut fungsi khusus. Berikut ini akan dibahas beberapa fungsi tersebut.

1. Fungsi konstan

Suatu fungsi f : x -> c dengan c ∈ R disebut fungsi konstan

Contoh:

Fungsi f (x) = 2

Fungsi f (x) = a

2. Fungsi identitas

Suatu fungsi f : x -> x (fungsi yang memtakan setiap anggota domain ke dirinya sendiri) disebut fungsi identitas.

Contoh:

Fungsi f(x) = x

Pemetaan ((1,1), (2,2), (3,3))

3. Fungsi linear

Suatu fungsi f : -> ax + b dengan a,b ∈ R disebut fungsi linear

Grafik fungsi linear berupa garis lurus

4. Fungsi kuadat

suatu fungsi f : x -> ax2 + bx + c dengan a,b,c ∈ R disebut fungsi kuadrat berbentuk parabola.

Contoh:

Fungsi f (x) = x2 + 4

5. Fungsi nilai mutlak

Catatan : IxI = x untuk x ≥ 0

-x untuk x < 0

6. Fungsi tangga

Jika IxI menyatakan bilangan bulat terbesar yang besar atau sama dengan x, maka fungsi f : x -> IxI disebut fungsi tangga.

7. Fungsi genap dan fungsi ganjil

- Suatu fungsi y = f (x) disebut fungsi genap apabila grafiknya simestris terhadap sumbu Y, yaitu

f (-x) = f (x) untuk semua bilangan x ∈ D f

- Suatu fungsi y = f (x) disebut fungsi ganjil apabila grafiknya simestris terhadap titik asal O (0,0), yaitu

f (-x) = -f (x) untuk semua bilangan real x ∈ D f

Komposisi Fungsi

Komposisi fungsi adalah suatu metode untuk menggabungkan fungsi. Metode ini berlandaskan pada proses aljabar yang sudah umum yaitu subtitusi. Sebagai contoh perhatikan fungsi f dan g yang di definisikan oleh:

f (x) = x2 dan g (x) = 3x +1

Pilih sembarang bilangan didalam domain fungsi g, misalkan x = -2 kita hitung

g (-2) = 3(-2) + 1 = -5

Hasil -5 dari g diproses lagi menjadi masukan untuk fungsi f . diperoleh f (-5) = (-5)2 = 25. Proses ini ditulis sebagai f(g(-2)) = 25. Berdasarkan uraian tersebut dapat disimpulkan:

a. Mulai dengan memasukan nilai x ke fungsi g dan hitunglah g (x)

b. Gunakan hasil g (x) sebagai suatu masukan bagi fungsi f dan hitunglah nilai f (g(x)). Hasil f (g(x)) sering dinotasikan sebagai f o g dibaca “ f bundaran g “.

Komposisi fungsi f o g berdasarkan formula fungsi f dan g adalah:

( f o g )(x) = f [ g(x) ]

Domain dari (f o g) terdiri atas masukan x dan g (x) ∈ domain f

Invers Fungsi

Perhatikan gambar di bawah ini!

Fungsi f : A -> B menyatakan pemetaan setiap a ∈ A ke b ∈ B dengan f(a) = b, sebaliknya adakah fungsi g : B -> A sedemikian sehingga g(b) = a ?

Jika fungsi g tersebut ada maka fungsi g disebut invers dari fungsi f dan fungsi f adalah invers dari fungsi g. Dua fungsi f dan g saling invers satu sama lainnya apabila memenuhi:

f[g(x)] = x untuk semua x dalam domain g, dan

g[f(x)] = x untuk semua x dalam domain f

berlaku pula sifat:

f[f-1(x)] = x untuk semua x dalam domain f-1, dan

f-1[f(x)] = x ntuk sema x dalam domain f

Contoh:

Tunjukkan bahwa fungsi f dan g berikut ini saling invers ( g= f-1 dan f = g-1 )

f (x) = x – 2, g (x) = x + 2

Jawab:

f (x) = x – 2, g(x) = x + 2

(f ° g)(x) = f (g(x))

= f ( x + 2 )

= ( x + 2 ) – 2

= x

(g ° f)(x) = g (f(x))

= g (x – 2)

= ( x – 2 ) + 2

= x

Karena (f ° g)(x)= x dan (g ° f)(x) = x maka g = f-1 dan g

= f-1 dan f = g-1 saling invers
